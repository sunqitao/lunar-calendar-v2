# Task-001: 产品设计分析 - 苹果风格参考文档

## 🎯 任务目标
分析苹果日历应用的设计特点，输出完整的设计参考文档，为 UI 设计和前端开发提供指导。

## 📋 工作内容

### 1. 苹果设计语言分析
- 研究 Apple Human Interface Guidelines
- 分析 iOS 日历应用的视觉风格
- 总结设计原则（简洁、层次、留白）

### 2. 配色方案定义
- **主色调**：iOS 系统蓝 (#007AFF)
- **辅助色**：系统灰、白色、黑色
- **强调色**：农历节日、节气标识色
- **深色模式**：适配 iOS 深色主题

### 3. 字体规范
- 中文字体：PingFang SC（苹方）
- 英文字体：SF Pro Display
- 字号层级：标题、正文、辅助信息

### 4. 布局规范
- 日历网格布局（7 列）
- 日期卡片设计
- 农历信息显示方式
- 节气、节日标识位置

### 5. 交互动画
- 页面切换动画（60fps）
- 日期选择反馈
- 滚动惯性效果

## 📤 交付物

1. **设计参考文档** (`design-reference.md`)
   - 配色方案（色值、使用场景）
   - 字体规范（字号、字重、行高）
   - 组件库（按钮、卡片、导航）
   - 布局示例

2. **功能优先级列表** (`feature-priority.md`)
   - P0: 核心日历功能
   - P1: 农历、节气显示
   - P2: 节日标记、自定义事件
